## 网易云音乐 - uniapp

### 平台兼容性
✅ H5
✅ 微信小程序
✅ Android
✅ iOS
⭕️ 其他平台未测试

### 支持的功能
1. ~~密码登录~~、~~手机验证码登录~~、二维码登录、游客登录
2. 搜索歌曲、歌单、电台
3. 我的歌单
4. 云盘
5. 添加、移除喜欢
6. 添加歌曲到歌单、把歌曲从歌单移除

### 接口
使用 [NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi) 私有部署
